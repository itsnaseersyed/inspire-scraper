About the Project

The INSPIRE Contact Details Scraper is an open-source initiative designed to democratize access to public educational data.

Technical Architecture

Built using Python and Flask, the application leverages:

  • BeautifulSoup4 for HTML parsing
  • Requests for efficient HTTP sessions
  • Pandas for structured data manipulation and CSV export
  • Vanilla JS for a lightweight, responsive frontend
